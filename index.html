Esculea de código PILRES

El objetivo de la Escuela de código de PILARES es contribuir a la participación principalmente de las mujeres en el mundo de la programación, en particular como programadoras WEB, y su inserción en instituciones, corporaciones o bien de forma independiente. La Escuela de código es una propuesta con perspectiva de género. Actualmente la Escuela de código ofrece dos módulos: Conociendo el ambiente de computación Introducción al cómputo y a sistemas Linux Elaboración de sistemas web Conocimientos para poder elaborar páginas (estáticas y dinámicas) y aplicaciones web. Las participantes de la Escuela de código adquirirán conocimientos en: Sistemas operativos Linux Git Programación web Front-end: HTML, CSS, JavaScript, Bootstrap, DOM Back-end: Node.js, Express.js, Sails.js, PHP, Laravel, MongoDB, SQLite, REST-API, MVC.
